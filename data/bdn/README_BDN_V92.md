# BDN livestock density layer for vet.ector

This folder contains a prototype BDN-derived livestock density layer.

Important:
- This is not an outbreak source.
- Use BDN only as denominator/exposure context: species, region, province, farms, heads, density level.
- Replace demo values with authorized or public aggregated BDN/VetInfo statistics before production use.

Supported species keys:
- bovine
- swine
- ovine
- caprine
- equine
- poultry
